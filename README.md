# CalCo Prints Software

Digital workspace for the CalCo Prints website, ecommerce planning, product catalog templates, and future lightweight business tools.

## Live Website

The public website is served from `index.html` with GitHub Pages.

`Website/index.html` is kept as the working copy inside the website folder. Keep the root `index.html` in sync before publishing updates.

## Folder Contents

- `Website/` - static website files.
- `Website-and-Commerce-Plan.md` - early website and selling plan.
- `Product-Catalog-Template.md` - product listing structure.
- `Automation-Ideas.md` - future automation notes.
- `CONTEXT.md` - routing and operating notes for this workspace.
