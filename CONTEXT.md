# Software Context

This workspace holds digital systems for CalCo Prints.

Use this folder for:

- Website planning.
- Ecommerce setup.
- Product listings.
- Inventory tracking.
- Order tracking.
- Automation.
- File organization.
- Photo workflow.
- Simple business tools.

Do not use this folder for:

- Printer maintenance logs.
- Tax records.
- Raw brand assets.
- Product design specs unless needed for website listings.

## Read First

- `Website-and-Commerce-Plan.md`
- `Automation-Ideas.md`

## Software Principle

Start simple. Build digital systems only when they reduce manual work or prevent mistakes.

Good early systems:

- Product catalog.
- Inventory tracker.
- Sales tracker.
- Expense tracker.
- Product listing copy bank.
- Photo naming workflow.

Avoid too early:

- Complex custom ecommerce.
- Heavy automation before the workflow is stable.
- Multiple disconnected trackers.
